## 🧭 Sobre mí

Soy arquitecto de software con base sólida en sistemas empresariales, microservicios y diseño técnico. En esta etapa estoy enfocando mi perfil de forma más intencional hacia **DevOps / Platform Engineering**, con una aproximación práctica y aterrizada:

- automatización de builds, despliegues y flujos CI/CD
- contenedores y estandarización de entornos
- Kubernetes on-premise y despliegues con Kustomize
- observabilidad, operación y mejora continua
- Linux, terminal y tooling útil para el día a día

> Este perfil está en constante evolución. Mis repositorios públicos visibles reflejan parte de mi trayectoria; mi foco actual está cada vez más orientado a DevOps, plataforma y automatización técnica.

## 🔎 En qué estoy poniendo el foco

| Área | Enfoque actual |
|---|---|
| CI/CD | GitLab CI/CD On-Premises, runners, pipelines reproducibles, versionado y despliegues controlados |
| Containers | Docker, optimización de imágenes, hardening básico y flujos de build consistentes |
| Kubernetes | despliegues on-premise, overlays con Kustomize, ingress, configuración y operación |
| Observabilidad | Prometheus, Grafana, health checks, métricas y alertas con valor operativo |
| Plataforma | estandarización técnica, mejoras transversales y documentación técnica |
| Desarrollo | Java, Spring Boot, microservicios y arquitectura como soporte para decisiones operativas |
